<a id="chat"></a>
## 高级聊天与多模态（ChatGPT / Gemini）

- 语音模式
  - 用高级语音模式先口述 PPT 思路，再让 AI 生成结构大纲
- NotebookLM
  - 把资料丢进去，让它生成思维导图和概念 PPT
- 协作画布（Canvas）
  - 用 Canvas 编写 SOP 或图文汇报，练习用模板快速排版
- Nano Banana
  - 给女儿写童话科普图，准备 3 个关键词+画风提示即可
- 练习
  - 选择一份最近的汇报，试着用语音+Canvas 在 30 分钟内产出初稿

<a id="cli"></a>
## 命令行助手（CLI）

- 为什么值得学
  - CLI 可以批量处理文件，省去一堆重复操作
- 安装 Gemini-cli
  - 按步骤装好后，先跑 `--help` 熟悉命令
- 实用场景
  - 批量改文件名：把一批照片按「日期-主题」命名
  - 转换文件格式：把 `.docx` 汇总成 `.pdf`/`.md`
  - 操作表格：用命令行整理一份 Excel 名单
- 练习
  - 挑一批零散文件，试着用 Gemini-cli 命令完成重命名

<a id="code"></a>
## AI 编程工具

- 适合设计师/非科班
  - 你只需要会描述需求，AI 会生成代码雏形
- 安装并熟悉 Google Antigravity
  - 跟着指南跑一遍 demo，了解界面和 Code Agent
- 打好基础
  - 用 git 做版本管理，学会 `commit` 和切换分支
- 实战思路
  - 让 Code Agent 生成一个静态网页（比如作品集封面），再自己微调样式
- 练习
  - 选一个网页灵感，尝试用 Agent 生成第一版，然后在浏览器里调色/排版
