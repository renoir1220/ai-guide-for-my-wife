# 什么是命令行 AI 工具，为什么重要？

最近，ChatGPT 在庆祝自己的三周年。三年了，我们与 AI 交互的主要方式仍然还是通过一个对话框。

仔细想想，这像不像我们小时候学的 DOS 界面？你可以认为跟 AI 对话，就像在跟计算机的 DOS 系统打命令一样，只不过 AI 足够聪明，能够把你的任何自然语言都理解为命令。

![DOS 界面与 AI 对话框的对比](/images/dos_vs_ai.png)

那么，如果我们真的能像跟 AI 对话一样，跟我们的电脑去对话，这将产生什么样的可能性呢？

*   比如我有一整个文件夹的火车票、发票文件，它们的文件名都是一串没有意义的数字。我能不能要求电脑把这些发票文件都命名为 `出发地-到达地-日期-金额.PDF` 这样的格式呢？
*   比如我能不能要求我的电脑把一段视频的第 30 分钟到第 60 分钟的音频提取为 1 个 MP3？
*   比如我能不能把一个文件夹以每一个人的月度总结，快速汇总为一个部门的月度总结？
*   我能不能把文件夹里的图片按照大小的范围放到不同的文件夹里，然后在根目录提供一个我指定格式的 Excel 去为这些文件做一个索引？

如果我们把这样的命令发给 AI 的网页对话框，我们是很难做到这一点的，因为它无法直接操作你的本地文件。

**但是通过命令行的 AI 工具，我们就可以把 AI 能为我们做的事情，扩展到我们的电脑系统里。**

我们可以让 AI 访问我们的系统文件，操作我们的文件，调用我们安装好的软件。

## 智能体 (Agent)：AI 的下一形态

这些命令行的 AI 工具，它们的强大远不止于此。它们其实就是现在非常流行的一个词叫 **Agent**（智能体）。

它们会根据你任务的复杂程度去做计划，然后像人一样去操作你的电脑，执行这些计划。

为了完成这些计划，它们不仅可以调用你系统中的工具，甚至还可以**自己去安装新的工具**。
*   比如说为了阅读 PDF 的内容，它们会安装 PDF 的阅读工具；
*   为了实现视频提取音频并做切割，它们会去下载相关的 `ffmpeg` 等工具；
*   为了能够读取和写 Word 文档，它们也会去获取相关的库。

<Callout title="核心概念：智能体 (Agent)">
  这样的 Agent 把 AI 的能力带入到了下一个时代。
  
  如果你的任务适合用这样的 Agent 完成，而你还在用对话的方式在网页端去复制粘贴，那么当你掌握了这样的 Agent 工具之后，你的效率可以提升数倍。

学习这样的工具，在目前是有一些门槛的。因为命令行本来就不是人人都会去用。而且，为了让这样的 Agent 能够运行，我们还需要去安装一些配套的环境。

但我觉得这样的投入是值得的。
1.  **生产力暴增**：尤其是大量重复性的工作，可能会从几小时变成瞬间完成。
2.  **未来技能**：如果操作系统不变成一个面向方便 AI 操作的系统设计（显然，这是在短期内无法实现的），那这种命令行的方式就不会快速变为“傻瓜式”，所以这个技能不会很快就变得没有意义。

## 安装 Gemini-CLI

接下来我们以安装 `gemini-cli` 为例，为大家介绍如何安装和使用命令行 AI 工具。

别看这些步骤挺简单，但是因为我们的电脑很少是新电脑，环境各异，在你安装这些内容的时候，很有可能会遇到一些“坑”。

> **经验之谈**：这个时候一定要静下心来去看那些报错。你可以把它们复制出来，去问 GPT 怎么解决这个问题。
> 
> 我觉得应该有一个 **1 小时**的心理预期，从你开始安装到真正能够与 Gemini 在命令行中对话。

### 步骤 1：安装 Node.js

很多 AI 工具依赖 Node.js 环境。

1.  访问 [https://nodejs.org](https://nodejs.org)
2.  下载 **LTS** 版本（长期支持版），点击安装即可。

![Node.js 官网下载页面](/images/nodejs_download.png)

### 步骤 2：安装 gemini-cli

打开你的终端（Terminal 或 PowerShell），输入以下命令并回车：

```bash
npm install -g @google/gemini-cli
```

### 步骤 3: 登录 Google 账户

在终端运行以下命令启动 Gemini，并按照提示登录你的 Google 账户：

```bash
gemini
```

## 使用 Gemini-CLI

接下来就可以开始去使用 `gemini-cli` 了，你可以像跟网页版的 AI 对话一样，跟它说话。

**不同的是，它可以访问你的本地资源。**

比如你在某个文件夹上右键打开终端，然后运行 `gemini`，你可以问它：

> “看一看这个文件夹里面都有什么内容？”

我相信你可以通过这样一个简单的动作，快速 Get 到它能做什么。

![在终端中使用 Gemini 查看文件夹内容](/images/gemini_cli_demo.png)

我们会在后面的章节讲解一些实际的使用方式，并通过实际的使用案例来介绍更多概念。
